# Pseudorandom Delivery

## Description
This script calculates a list of pseudorandom ID's (as consecutive numbers) of buildings, to which a delivery should be sent during one day. The user should provide:
* name of the report file,
* value of the lowest delivery count per day,
* value of the highest delivery count per day,
* number of buildings.

## Usage and results
Run script in your favourite Python IDE. Commands to interact will be shown in terminal/shell. The report file (*.txt) will be saved in the same directory as Python script.

## Requirements
Python 3.x
